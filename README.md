# Line-Follower-Robot-Arduino
Autonomous line follower robot using Arduino that detects path using sensors and controls motor movement through a motor driver for intelligent navigation.

##  Hardware Connections

### IR Sensors
| Sensor | Arduino |
|--------|----------|
| Left | D2 |
| Right | D3 |

### Motor Driver (L293D)
| Pin | Arduino |
|-----|----------|
| IN1 | D9 |
| IN2 | D10 |
| IN3 | D11 |
| IN4 | D12 |

### Motors
Connected to L293D output pins
